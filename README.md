# Nayyab Jewellers (Vite + React)

Modern jewellery storefront built with Vite and React.

## Features

- Hero landing section with premium branding
- Product collection with category filters
- Add-to-cart flow for creating order list
- WhatsApp direct order and cart checkout links
- Customer reviews and trust highlights
- Mobile responsive design

## Getting Started

```bash
npm install
npm run dev
```

Open the local URL shown by Vite (usually `http://localhost:5173`).

## Build for production

```bash
npm run build
npm run preview
```
